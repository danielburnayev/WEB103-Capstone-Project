# User Stories

Reference the Writing User Stories final project guide in the course portal for more information about how to complete each of the sections below.

## Outline User Roles

[👉🏾👉🏾👉🏾 Include at least at least 1, but no more than 3, user roles.]

### Roommate: 
A user who lives in a shared household and participates in the household by completing chores, adding items, and contributing to shared responsibilities.

## Draft User Stories

[👉🏾👉🏾👉🏾 Include at least at least 10 user stories in this format:]

1. As a [user role], I want to [what], so that [why].

1. As a [roommate], I want to [view a chore list], so that [I know what tasks need to be completed].
2. As a [roommate], I want to [mark chores as completed], so that [my other roommates know the task has been finished].
3. As a [roommate], I want to [add items to a shared grocery list], so that [everyone knows what needs to be purchased].
4. As a [roommate], I want to [mark grocery items as purchased], so that [my roommates can avoid buying duplicates].
5. As a [roommate], I want to [create shared bills], so that [expenses can be tracked within the household].
6. As a [roommate], I want to [view how much I owe on a bill], so that [I can pay my share on time].
7. As a [roommate], I want to [create or join a household group], so that [I can collaborate with the people I live with].
8. As a [roommate], I want to [add and view fridge items], so that [I can track available food, plan meals, and reduce waste].
9. As a [roommate], I want to [mark a bill as paid], so that [my roommates know I have completed my payment].
10. As a [roommate], I want to [add a comment on chores], so that [I can communicate updates or issues with my roommates].
