# Entity Relationship Diagram

Reference the Creating an Entity Relationship Diagram final project guide in the course portal for more information about how to complete this deliverable.

## Create the List of Tables

User: A table for storing all the users that are guests or have signed up to use the website.

Calendar User: A table for storing all the users identities in select calendars, throughout all the calendars.

Calendar: A table for storing all the calendars that users place events in to share their availability with others.

Event: A table for storing all of the events that have been made in calendars.
